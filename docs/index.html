#Introduccion

Que es lienzo?

Lienzo es un game engine que esta dise√±ado para facilitar el desarrollo de videojuegos con una arquitectura basada en entidades de juego y componentes.

Cuales herramientas lienzo brinda para el desarrollo de video juegos?

Lienzo brinda un conjunto de herramientas para facilitar el desarrollo de videojuegos como un sistema de eventos un motor de renderizado un motor de fisicas y un sistema de UI.

Porque deberia utilizar lienzo?

Lienzo es muy facil de utilizar ademas de que el marco de trabajo es similar al de la mayoria de game engines populares como unity o unreal.

#Escena

La escena es lo primero que debes configurar antes de empezar a crear entidades del juego o gameObjects. para crear una scene debes: 
              const scene = new lienzo.Scene({
                gravity: new Vector(0, 10), // Establece la gravedad que tendra el sistema de fisicas
                system: 'Rect', // Establece el sistema de fisicas que sera utilizado
                backgroundColor: '#9FE6FF', // Establece el color con el cual se limpiara la pantalla
                bounds: [6000, 0, 500, 0] // Bordes del sistema de fisicas
              })
Luego para utilizar ese escena debes: lienzo.manager.setScene(scene)

#Entidad de juego o GameObject

Como crear gameObjects? Para crear un game object puedes utilizar: 

                  const character = new lienzo.gameObject({
                    name: 'character',
                    position: new Vector(percent(50, 50)[0], percent(50, 50)[1]),
                    scale: new Vector(3, 3),
                    collider: 'fit',
                    spriteSheet: {
                      src: 'assets/walk.png',
                      size: new Vector(16, 28)
                    }        
                  })
                  scene.add(character)

#Componentes

Los componentes son las partes que componen a un object de juego y estos pueden ser de varios tipos por ejemplo 

                    const character = new lienzo.gameObject({
                      name: 'character', // Name component. es utilizado para realizar busquedas en la lista de objetos de juego.
                      position: new Vector(percent(50, 50)[0], percent(50, 50)[1]), // Transform component
                      scale: new Vector(3, 3), // Transform component
                      collider: 'fit', // Collider component
                      spriteSheet: { // Sprite component
                        src: 'assets/walk.png',
                        size: new Vector(16, 28)
                      }        
                    })
                    scene.add(character)

#Sistema de eventos

El sistema consiste de la sigiente lista de eventos

  • mouseDown
  • mouseUp

  • keyUp
  • keyPress
  • globalMouseDown
  • globalMouseMove
